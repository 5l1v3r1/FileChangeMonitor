Never miss a feature update again.

File Change Monitor provides continous monitoring for JavaScript files hosted on websites running bug bounty programs. This way, you'll get an email whenever a change occurs in a website you're monitoring, so you can stay on top of the newest features.

View FileChangeMonitor on GitHub.

Features

  • Poll JavaScript files continuously for changes

  • Extract relative urls in strings from JavaScript files

  • Monitor files with dynamic names for changes in the filename

  • Receive an email when new relative urls are discovered
